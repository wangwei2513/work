### 开发环境：
Node.js + MongoDB + Express 4.x

### 如何使用：
* 首先启动MongoDB服务
* `npm install` 进行依赖模块安装
* 安装完成后，`npm start` 启动应用，浏览器打开 `http://localhost:3000` 即可预览
